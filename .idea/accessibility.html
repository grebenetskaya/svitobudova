Accessibility Statement

We believe worldbuilding should be accessible to everyone, regardless of physical or cognitive ability.

Compliance Status

This website is designed to be WCAG 2.2 Level AA compliant. Key features include:

  • Skip links: Keyboard users can bypass the navigation and go directly to the main content.
  • Readable contrast: The design uses dark text on light backgrounds and clearly visible buttons.
  • Semantic HTML: The pages use headings, navigation landmarks, main content areas, lists, forms, and tables correctly.
  • Keyboard navigation: Links, buttons, form fields, and menus can be reached and used with the keyboard.
  • Form labels: Input fields have visible labels and required fields are marked.
  • Reduced decorative effects: The website does not rely on animations, canvas effects, or decorative backgrounds to communicate information.

Known Limitations

This is an academic demonstration project. The checkout and contact forms are visual prototypes and do not submit real data.
